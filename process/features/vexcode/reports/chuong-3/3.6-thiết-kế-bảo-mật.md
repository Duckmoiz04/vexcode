# Chương 3: Phân tích và thiết kế hệ thống

## 3.6. Thiết kế bảo mật

Hệ thống không xử lý dữ liệu nhạy cảm của người dùng ngoại trừ API key của AI provider. API key được lưu trong file .env cục bộ, không được gửi qua mạng và chỉ được sử dụng bởi Python engine khi gọi AI API.

Cơ chế path safety được áp dụng cho tất cả đường dẫn đầu vào. Hàm `isPathSafe()` kiểm tra đường dẫn nằm trong workspace boundary, chống path traversal bằng case-insensitive prefix check.

Trước khi apply bất kỳ bản sửa nào vào file, hệ thống tạo bản backup tại thư mục `~/.vexcode/backups/`. Người dùng có thể restore file gốc từ backup bất cứ lúc nào.
