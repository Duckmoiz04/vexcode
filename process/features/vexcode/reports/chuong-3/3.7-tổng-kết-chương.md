# Chương 3: Phân tích và thiết kế hệ thống

## 3.7. Tổng kết chương

Chương này đã trình bày các yêu cầu chức năng và phi chức năng của hệ thống, khảo sát các công cụ Code Review hiện có và bài học rút ra, kiến trúc ba tầng với Node.js, Python và React, thiết kế chi tiết từng thành phần, pipeline tám giai đoạn (Scan → Enrich → Complexity → Dedup → Classify → Naming Audit → AI Resolve → Report), REST API, lưu trữ dữ liệu dạng JSON, và cơ chế bảo mật. Kiến trúc được thiết kế với graceful fallback ở mọi tầng để đảm bảo hệ thống luôn hoạt động ngay cả khi một số thành phần gặp lỗi.
