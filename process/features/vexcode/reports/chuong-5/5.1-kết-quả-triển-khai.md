## 5.1. Kết quả triển khai

Hệ thống VexCode được triển khai hoàn chỉnh với ba packages chính, tổng cộng hơn 15.000 dòng mã nguồn trên 160 file.

**Bảng 5.1.** Kết quả triển khai theo package

| Package | Ngôn ngữ | File mã nguồn | File test | Dòng code | Trách nhiệm |
|---------|----------|--------------|-----------|-----------|-------------|
| `packages/engine` | Python 3.12 | 28 | 19 | ~4.000 | Pipeline phân tích (scan, enrich, dedup, resolve, report) |
| `packages/cli` | Node.js ESM | 23 | 5 | ~3.000 | CLI entry point, Express REST API, bridge subprocess |
| `packages/web` | TypeScript + React 19 | 110 | 32 | ~8.000 | Web Dashboard (SPA), Sigma.js graph, Tailwind v4 |

Hệ thống hỗ trợ 4 AI providers (OpenAI, Anthropic, Google, 9router) thông qua cơ chế cấu hình động trong `.env`. Bộ phân loại ISO/IEC 25010 hỗ trợ 4 danh mục Phase 1 (Security, Reliability, Maintainability, Performance Efficiency) với hơn 50 mapping từ CWE ID và hơn 100 mapping từ rule_id keywords. GitNexus Knowledge Graph enrichment cung cấp symbol context, callers và blast radius với ThreadPoolExecutor 10 workers song song.

**Mục tiêu 1** — Deep Context-Aware Analysis: Module Code-Context Engine triển khai cơ chế lập chỉ mục AST qua GitNexus CLI, truy xuất ngữ cảnh symbol (callers, blast_radius), và bổ sung vào mỗi finding. Pipeline enricher sử dụng 10 parallel workers để gọi GitNexus context/impact commands cùng lúc.

**Mục tiêu 2** — Đánh giá đa tiêu chuẩn và đề xuất tối ưu: Pipeline đa-scanner (OpenGrep + Gitleaks + OSV) phát hiện lỗ hổng bảo mật + maintainability + operability; AI Resolution 3-stage (Analyze → Fix → Review) phân loại confirmed/false_positive và đề xuất fix code; Dashboard hiển thị kết quả trực quan gồm findings list, dependency graph (Sigma.js), dataflow trace, và chat AI.
