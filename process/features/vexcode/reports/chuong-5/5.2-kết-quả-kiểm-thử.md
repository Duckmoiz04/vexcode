## 5.2. Kết quả kiểm thử

### 5.2.1. Kiểm thử tầng CLI

Tầng CLI có 5 test files với tổng số 107 tests. Các test được viết bằng Vitest và bao phủ các module chính.

Server tests (63 tests trong server.test.js) kiểm tra tất cả REST API endpoints: GET/PUT config, GET scan stream, POST scan, POST cancel, GET reports, GET report detail, POST re-resolve, POST apply fix, POST chat, GET file-content, GET/POST backup. Các test sử dụng supertest để gọi HTTP và kiểm tra response status, JSON body, và error handling.

Bridge tests (15 tests) kiểm tra Python subprocess spawning, stdout/stderr parsing, scan cancel, và graceful fallback khi Python không khả dụng.

CLI tests (5 tests) kiểm tra command line argument parsing cho analyze, serve, và ui commands.

File service tests (4 tests) kiểm tra path safety với các trường hợp path traversal, đường dẫn tuyệt đối, đường dẫn tương đối, và đường dẫn hợp lệ. Finding status tests kiểm tra tracking trạng thái finding (open, in_progress, resolved, dismissed).

### 5.2.2. Kiểm thử tầng giao diện

Tầng giao diện có 32 test files với tổng số 224 tests. Các test được viết bằng Vitest với jsdom environment và React Testing Library.

Context tests kiểm tra AIProviderContext: khởi tạo state mặc định, cập nhật findings, cập nhật config, lọc findings, xử lý scan complete.

Hook tests kiểm tra useScan (SSE handling, cancel scan, progress tracking), useConfig (load config từ API, update config), useReports (load projects, load reports, load report detail), useChat (chat history, gửi câu hỏi), useToast (hiển thị và tự động ẩn toast), useAutoScroll (scroll đến element).

Component tests kiểm tra rendering và user interaction cho Header, Sidebar, SettingsDrawer, Dashboard components, Code Inspector, FilterPanel, ScanModal, Markdown renderer, ErrorBoundary.

Utility tests kiểm tra reportNormalizer (chuyển đổi report format), sarifAdapter (parse SARIF format), syntaxHighlight (highlight code blocks).

### 5.2.3. Kiểm thử tầng engine

Tầng engine có 19 test files. Các test được viết bằng pytest.

ISO 25010 taxonomy tests kiểm tra bộ phân loại với các trường hợp: CWE ID mapping, rule_id keyword mapping, OWASP ID normalization, context override cho style/naming convention, edge cases (CWE không tồn tại, rule_id rỗng, message null).

Scanner tests kiểm tra fast scan detection (git status parsing, không có git repo, repo sạch, có file thay đổi), và Opengrep scan execution.

AI Resolver tests kiểm tra prompt building, API call với mock responses, retry logic, parallel execution, và mock fallback.

Pipeline tests kiểm tra tích hợp giữa các phase: scanner + enricher, enricher + resolver, resolver + reporter.

AST Graph tests kiểm tra GitNexus CLI giao tiếp: availability check, symbol resolution, context fetching, impact computation, và mock fallback.
