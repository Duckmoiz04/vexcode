# Chương 4: Triển khai thực nghiệm

## 4.1. Tổng quan triển khai

Hệ thống AI Code Review được triển khai với ba packages chính: packages/cli (Node.js), packages/engine (Python), và packages/web (React). Mỗi package có cấu trúc module riêng và giao tiếp với nhau qua REST API hoặc JSON file. Chương này trình bày chi tiết cách triển khai từng thành phần, từ tầng CLI và REST API đến engine phân tích và giao diện người dùng.
