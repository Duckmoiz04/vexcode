## 4.6. Triển khai bảo mật

Hệ thống triển khai path safety kiểm tra mọi đường dẫn đầu vào. Hàm `isPathSafe()` sử dụng case-insensitive prefix check để ngăn path traversal attacks. Tất cả các API nhận đường dẫn đều gọi hàm này trước khi xử lý.

Backup được tạo trước mỗi lần apply fix. Backup lưu tại `~/.vexcode/backups/` với tên file bao gồm timestamp và tên file gốc. User có thể restore file từ backup bất kỳ lúc nào qua API.

AI API keys được lưu trong file .env cục bộ, không bao giờ được gửi qua mạng đến frontend. Khi trả về config cho frontend, API key được mask (chỉ hiển thị 4 ký tự cuối).
